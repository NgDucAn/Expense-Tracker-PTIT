Mô tả: Đây là màn hình UI của chức năng xem chi tiết một giao dịch. Màn hình này được hiển thị khi ấn vào 1 giao dịch trong danh sách các giao dịch của màn hình TransactionScreen.
Yêu cầu: 
Dựa vào hình ảnh đó, tạo một giao diện UI giống y hệt. 
Tạo một package là detailtransaction trong package ui.
Trong package detailtransaction thì tạo 3 file: 
File 1: DetailTransactionContract.kt chứa state, intent, event. Và các class này đều phải impliment interface MviStateBase, MviEventBase, MviIntentBase.
File 2: DetailTransactionViewModel sẽ extends BaseViewModel<MviStateBase, MviEventBase, MviIntentBase> và inject GetTransactionByIdUseCase, SaveTransactionUseCase.
File 3: DetailTransactionScreen.kt sẽ dùng để thiết kế UI.
Trong khi thiết kế UI nếu cần tách các components ra thì tạo package components ngang hàng với DetailTransactionScreen.
Tách thành hai hàm composable, một hàm là DetailTransactionScreen để gọi viewmodel và các hàm liên quan, một hàm là DetailTransactionContent với tham số là state của màn hình và các hàm liên quan, cùng với đó là tạo preview cho hàm DetailTransactionContent. 


Đảm bảo thiết kế UI và code đúng kiến trúc hiện tại của Project.
